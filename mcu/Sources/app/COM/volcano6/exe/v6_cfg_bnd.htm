V6BND: History


V6BND version 9.16:

  • Target added: "DIAB MPC57xx". FlexCAN pin configuration attributes removed. Pin configuration moved to target.c.

V6BND version 9.15:

  • Target added: "COSMIC STAR12 BANKED"

V6BND version 9.14:

  • Target added: "METROWERKS S12X BANKED CONST GLOBAL 128 IMF"
  • msCAN12S controller can support 128 immediate frames with the new target. (#00984141 enhancement)

V6BND version 9.13:

  • Configurable clock prescalers are supported in NECUARTD controller. (#00981638)

V6BND version 9.12:

  • Configurable wake up pulse length is supported in NECUARTF controller. (#00955837)

V6BND version 9.11:

  • Configurable wake up pulse length is supported in LINFlex controller. (#00955837)
  • AFCAN now supports 64 slots.

V6BND version 9.10:

  • Configurable wake up pulse length is supported in NECUARTD and SCI12LIN controllers. (#00955837)

V6BND version 9.09:

  • TL_Frame Definition updated.

V6BND version 9.08:

  • Controller added: "FlexCAN_gmask"
  • Target added: "DIAB MPC56xx"

V6BND version 9.07:

  • Controller added: "AFCAN_78K0R", "NECUARTF"
  • Target added: "IAR R8C FARDATA", "IAR 78K0RFX3"

V6BND version 9.06:

  • Initial release for VTP 6.21.
  • LIN 2.1 slave support added.
  • Target added: "CGT TMS470"

V6BND version 9.05:

  • Target added: "H8S H8SX"

V6BND version 9.04:

  • Controller added: "M16CCAN"
  • Target added: "IAR R8C", "COSMIC STAR12X BANKED GLOBAL"

V6BND version 9.03:

  • Controllers added: "NECUARTD"

V6BND version 9.02:

  • Controllers added: "msCAN12S"
  • Targets added: "METROWERKS S12 BANKED"

V6BND version 9.01:

  • Initial release for VTP 6.2.
  • Controllers added: "LINFlex"
  • Controllers updated: "AFCAN" and "SCI12LIN"
  • Targets added: "GHS V850" and "DIAB PACE6"

V6BND version 8.04:

  • RCAN updated to VTP 6.1

V6BND version 8.03:

V6BND version 8.02:
